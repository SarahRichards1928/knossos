Working with Knossos

Getting started

  1. Install Knossos and download this Python script. This script converts your TIFF image data into Knossos readable data cubes.
  2. Execute the Python script in the directory containing the TIFF files. This will generate a folder containing the generated Knossos dataset.
  3. Open Knossos and load the dataset by using File > Choose Dataset.... A window opens. Select the dataset path and keep all other settings as they are.

How to use Knossos

Knossos displays 4 views on your data. The first three viewports represent the XY, XZ and YZ plane, respectively (upper left, lower left and upper right). They are also distinguishable by their frame color: red, green and blue.

The fourth viewport shows a 3D view with the current position inside your data.

Drag across any viewport to navigate through your data. When your mouse hovers over a viewport, you can press D and F to move orthogonally inside it.
