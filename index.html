What is Knossos

Knossos is a scientific program that is used for biological or medical studies. It visualizes 3D records taken by electronic microscopes by displaying each plane separately. Take a look at Knossos features to learn more about it.

Getting started

Download Knossos

Knossos is available for Windows, GNU/Linux and OS X. Since Knossos is an open-source program, we host our releases on Github. Head to our Github repository {% include fa.html icon="fa-external-link" %} to download the appropriate version of Knossos for your system.

Example datasets

If you want to take a look at Knossos, but need a dataset to try it out, you can download two example datasets here: small (54 MB), large (400 MB)

An image dataset can be loaded into Knossos by selecting its .conf file in File - Load Dataset...

Own datasets

If you have own image datasets, they will probably need to be converted into Knossos’ format. Download the knossos_cuber script and take a look at the Getting started section to learn how to do so.

Feel free to contact us if you need any assistance in setting up Knossos.
